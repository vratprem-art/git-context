# git-context: The $2 Dev Tool That Pays for Itself in One Use

Every AI coding session starts the same: copy file tree, find relevant files, paste recent git history. 

**git-context** does it in one command:

```bash
python3 git-context --files -o context.txt
```

What you get:
- Complete project tree (respects .gitignore)
- Git branch topology with remote info
- Last 20 commits with graph
- All source file contents with syntax highlighting
- Working tree status (staged/unstaged changes)

## Zero dependencies
Pure Python 3. No pip install needed. Just download and run.
